Intergration with Flat UI

Startup framework based on Flat UI Free version 2.0.0, so you can use all elements of this version inside it.

The same way you can buy the profesional version of the Flat UI framework and use it instead of Flat UI version 2.0.0.

In order to use Flat-UI Pro you need to put it in the 'flat-ui' folder in your project directory and remove all elements marked as "Using only with Flat-UI (free)" from the code.
