User Guide

First of all thanks for your purchase of Startup Framework. It's a great solution for those who want to change their startup website not wasting time. The Startup Framework contains a set of PSD, HTML, CSS and Java script code which you can test and choose something that you like for use it in your project.

Startup Framework contains components and complex blocks which can easily be integrated into almost any design. All of these components are made in the same style, and can easily be integrated into projects, allowing you to create hundreds of solutions for your future projects.
