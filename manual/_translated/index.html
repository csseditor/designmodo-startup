Startup

We have a great surprise for Designmodo fans:
our first free HTML user interface kit.

It contains many basic and complex components which are great for designers to have at hand: buttons, inputs, button groups, selects, checkboxes and radio-buttons, tags.

Overview

The Startup framework is a constructor with ready-made solutions - pieces of html, CSS and Java script code which you can try out and pick something that you would like to use in your project.

Startup framework is based on Flat-UI version 2.0.0, so you can use all the elements of this version inside it.

You can also buy the professional version of the Flat-UI framework and use it instead of Flat-UI version 2.0.0.

In order to use Flat-UI Pro you need to put it in the 'flat-ui' folder in your project directory and remove all elements marked as "Using only with Flat-UI (free)" from the code.

Startup framework has its own additional elements which makes your job even simpler.
