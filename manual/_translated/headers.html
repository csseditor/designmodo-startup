Header block. Codename: “header-1”

  1. Overview

    This block was developed as a special header block. All blocks that are marked as header should only be used once on one page. This header block has a navigation bar, which can have two states. The standard state is described further in this documentation and the fixed-top state is shown in the example for this for this block.

  2. Code

    Simple version:

    If you haven’t already done all the instructions from the how to install page of this documentation, please do that first.

    To use this block, copy and paste the following line of code into your HTML file: 

    ...

    "LESS" version:

    If you haven’t already done all the instructions from the how to install page of this documentation, please do that first.

    Copy the same code as in simple version and paste it in your '.less' file. Put the following line after that code: 

    @import '../../ui-kit-header/less/header-1.less';

    All versions:

    In this section you will see what modifications you can do to the block.

    This block, as many other header blocks in our framework, has a special place for background image, which is: <div class=”background”>...</div>. To put your own image as background you need to add CSS rules for this element. You can also manage the opacity for this element. Here is an example: 

    .background {
    background-image:url(‘your image url’);
    opacity: opacity value;
}

    In the “Less” version, you can use the cross browser function for the opacity property: 

    .opacity(value from 0 to 100);

    This block has a special functionality carousel. You can modify the content of the carousel slides by replacing the content inside <div class=”pt-page”> ... </div>. This block has a “.background” attribute for the slides, so you can change the background for any one of slides individually.

    This block was developed as a special header block. All blocks that are marked as header should only be used once on one page. This header block has a navigation bar, which can have two states. The standard state is described further in this documentation and the fixed-top state is shown in the example for this for this block. The navigation bar is hidden by default, but you can activate it by removing the class ".hidden" from the following element: <header class="header-17 hidden"> .

  2. Code

    Simple version:

    If you haven’t already done all the instructions from the how to install page of this documentation, please do that first.

    To use this block, copy and paste the following line of code into your HTML file: 

    ...

    "LESS" version:

    If you haven’t already done all the instructions from the how to install page of this documentation, please do that first.

    Copy the same code as in simple version and paste it in your '.less' file. Put the following line after that code: 

    @import '../../ui-kit-header/less/header-17.less';

    All versions:

    In this section you will see what modifications you can do to the block.

    This block, as many other header blocks in our framework, has a special place for background image, which is: <div class=”background”>...</div> To put your own image as background you need to add CSS rules for this element. You can also manage the opacity for this element. Here is an example: 

    .background {
    background-image:url(‘your image url’);
    opacity: opacity value;
}

    In the “Less” version, you can use the cross browser function for the opacity property: 

    .opacity(value from 0 to 100);

    You can replace the content in this block with your own. In order to do this you need to replace the content between the tags <div class="hero-unit">...</div> . This block also has a carousel. In order to replace the content on any of the slides place it between these tags: <div class=”pt-page”> ... </div> .
