We have a great surprise for Designmodo fans – our first free HTML user interface kit. Flat UI Free is made on the basis of Twitter Bootstrap in a stunning flat-style, and the kit also includes a PSD version for designers.

We have considered your needs in developing the html version of our UI Kit, which contains the same elements as the PSD version and are perfect for creating great websites. With this UI Kit, you can spend even less time on routine work, saving that time for creativity.

Flat UI Free contains many basic and complex components which are great for designers to have at hand: buttons, inputs, button groups, selects, checkboxes and radio-buttons, tags, menus, progress bars and sliders, navigation elements and more.

    "LESS" version:

    All versions:

    So about modifications which you can do.

    Carousel

    As you can see in code we have block with id="b1-pt-1" - it's main block of the carousel. In this block situated few pages whict have class="pt-page". For thouse elements you can change background, and you can do it in few ways. First way - you can change the part of class that start with bg- for this element. This way you can use reserved classes from Flat-UI, but it changed only color of carousels page background. Or second way - you can use css for change background for concrete element. As example for pt-page-1. You can do something like this: 

    .pt-page-1{
    background-image:url('some image url');
    background-color: some color;
}

    You can change text for any of this carousel pages. For this you should change text in block with class="hero-unit"

    And you can add new pages. For this you should copy one of blocks with class="pt-page", replace class as pt-page-1 to new class, and new class should has new number.

    Content except carousel

    You can replace any text except code structure.
