Blog Block #2

  1. Overview

    This block has a big image on the left which you can replace with any image you like. It also has a logo area and a content area. All of these components can be modified in any way you want.

  2. Installation

    If you havenâ€™t already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    All you need to do is copy and paste these lines of code between <body> ... </body> into your HTML file. 

    include "../../../ui-kit/ui-kit-blog/for-tpl/blog-2.html"

  4. LESS version

    Copy the same code as in css version and paste it in your '.less' file. Put the following lines after that code: 

    @import 'blog-common.less';
@import 'blog-2.less';

  5. Samples

    For this block we have created an example so you can see how it looks combined with other elements of our framework.

    You can see it in include "sample-b2-man.html".
