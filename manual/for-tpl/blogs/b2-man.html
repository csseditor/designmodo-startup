Block for creating blog. Codename "blog-2"

  1. Overview

    This block has big image in left part which you can replace to any picture that you want, block with logo, and content block. All of this components you can modificate as you want.

  2. Code

    Simple version

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you only need paste next rows between <body> ... </body> into your html file. 

    include "../../../ui-kit-blog/for-tpl/blog-2.html"

    "LESS" version

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import 'blog-common.less';   
@import 'blog-2.less';
