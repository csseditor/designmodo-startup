Blog Block #1

Startup user manual

  1. Overview

    This block has carousel in which you can paste images and slogan that you want, block with headline, and content block with menu. This block has special grid for content. All of this components you can modificate as you want.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    
include "../../../ui-kit-blog/for-tpl/blog-1.html"

  4. LESS version:

    You should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    
@import 'blog-common.less';   
@import 'blog-1.less';

  5. Component Modification

    So about modifications which you can do.

    As you can see in code we have block <div id="b1-pt-1">...</div> - it's main block of the carousel. In this block situated few pages <div class="pt-page">...</div> For thouse elements you can change background, and you can do it in few ways. First way - you can change the part of class that start with bg- for this element. This way you can use reserved classes from Flat-UI, but it changed only color of carousels page background. Or second way - you can use css for change background for concrete element. As example for <div class="pt-page-1">...</div> . You can do something like this: 

    
.pt-page-1{
    background-image:url('some image url');
    background-color: some color;
}

    You can change text for any of this carousel pages. For this you should change text in block <div class="hero-unit">...</div>

    And you can add new pages. For this you should copy one of blocks <div class="pt-page">...</div> , replace class as pt-page-1 to new class. Each new page should has new class with new number.

    You can replace any text except code structure.
