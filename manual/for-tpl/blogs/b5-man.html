Block for creating blog. Codename "blog-5"

  1. Overview

    This block has slogan on top of it. Row with special information as user, time and coments. All of this information items has specific icon. This block has a carousel functionality, which you can use as carousel with images and text that you want. An you can use it as carousel with banners. After this carousel text block with slogan as delimiter. And in bottom of this block special grid that you can use as you want. The last element of this block - tags as links.

  2. Code

    Simple version

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you only need paste next rows between <body> ... </body> into your html file. 

    include "../../../ui-kit-blog/for-tpl/blog-5.html"

    "LESS" version

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import 'blog-common.less';   
@import 'blog-5.less';

    All versions

    Carousel

    As you can see in code we have block <div id="b1-pt-1">...</div> - it's main block of the carousel. In this block situated few pages <div class="pt-page">...</div> For thouse elements you can change background, and you can do it in few ways. First way - you can change the part of class that start with bg- for this element. This way you can use reserved classes from Flat-UI, but it changed only color of carousels page background. Or second way - you can use css for change background for concrete element. As example for <div class="pt-page-1">...</div>. You can do something like this: 

    .pt-page-1{
    background-image:url('some image url');
    background-color: some color;
}

    You can add text for any of this carousel pages. For this you should add content after <div class="box-icon">...</div>. But you should create style for this text by yourself.

    And you can add new pages. For this you should copy one of blocks <div class="pt-page">...</div>, replace class as pt-page-1 to new class. Each new page should has new class with new number.

    Content except carousel

    You can replace any text except code structure.
