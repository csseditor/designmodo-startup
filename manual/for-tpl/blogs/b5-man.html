Blog Block #5

  1. Overview

    This block has slogan on top of it. Row with special information as user, time and coments. All of this information items has specific icon. This block has a carousel functionality, which you can use as carousel with images and text that you want. An you can use it as carousel with banners. After this carousel text block with slogan as delimiter. And in bottom of this block special grid that you can use as you want. The last element of this block - tags as links.

    If you havenâ€™t already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    All you need to do is copy and paste these lines of code between <body> ... </body> into your HTML file. 

    include "../../../ui-kit/ui-kit-blog/for-tpl/blog-5.html"

  4. LESS version:

    Copy the same code as in css version and paste it in your '.less' file. Put the following lines after that code: 

    @import 'blog-common.less';
@import 'blog-5.less';

  5. Component Modification:

    As you can see in code we have block <div id="b1-pt-1">...</div>, the main block of the carousel. There are a few page elements in this block: <div class="pt-page">...</div> You can change the background of those elements in a few different ways. One way is changing the part of the class that starts with bg- for this element. This way you can use reserved classes from Flat-UI, but it will only change the background color of the carousel page. Another way is to use CSS to change the background for a specific element. As an example for block <div class="pt-page-1">...</div>, you can do something like this: 

    .pt-page-1{
    background-image:url('some image url');
    background-color: some color;
}

    You can add text to any of the carousel pages by putting content after <div class="box-icon">...</div> . However, you need to create style for this text yourself.

    You can also add new pages to the carousel. In order to do this, you need to copy one of blocks <div class="pt-page">...</div> and change the class pt-page-1 to new class. Each new page should has new class with new number.

    You can replace any text in the block but you should maintain the HTML structure.
