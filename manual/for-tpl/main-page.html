Startup

We have a great surprise for Designmodo fans our first free HTML user interface kit.

It contains many basic and complex components which are great for designers to have at hand: buttons, inputs, button groups, selects, checkboxes and radio-buttons, tags.

Overview

The Startup Framework is a constructor with ready-made solutions - pieces of html, CSS and Java script code which you can test and choose something that you like for use it in your project.

Startup framework based on Flat-UI version 2.0.0, so you can use all elements of this version inside it.

The same way you can buy the profesional version of the Flat-UI framework and use it instead of Flat-UI version 2.0.0.

For use Flat-UI Pro you should place it into flat-ui folder inside the project and remove from code all elements that marked as "Using only with Flat-UI (free)".

Startup framework has own elements that makes you work simpler.
