Block for creating blog with carousel. Codename "blog-1"

  1. Overview

    This block has carousel in which you can paste images and slogan that you want, block with headline, and content block with menu, texts and images. All of this components you can modificate as you want.

  2. Code

    For Simple version: if you not already done all instructions that you had to read on the home page of this documentation please make it first you only need paste next rows between <body> ... </body> into your html file. 

    include "../../ui-kit-blog/for-tpl/blog-1.html"

    For "LESS" version: if you not already done all instructions that you had to read on the home page of this documentation please make it first you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import 'blog-common.less';   
@import 'blog-1.less';
