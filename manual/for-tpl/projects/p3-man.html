Block for showing projects. Codename “projects-3”.

  1. Overview

    This type of block - projects block was developed special for demonstrate your projects to users. It can be used as portfolio of designer or else... Any block with this type has special active areas with images which has hover effects. Functionality that will run after an user click on this elements you should implement by yourself.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit-projects/for-tpl/projects-3.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-projects/less/project-3.less';

    All versions:

    So about modifications which you can do.

    This block has slogon on top of it and block with three columns that consits special blocks for demonstrating your projects. You can modificate anyone of it, but you should remember that you can't reorganize the html structure if you want to leave functionality of this block as it exist now. For add images into project boxes you should insert your image between <div class="photo">...</div>.
