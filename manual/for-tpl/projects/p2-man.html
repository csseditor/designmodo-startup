Block Progects #2

  1. Overview

    This type of block - projects block was developed special for demonstrate your projects to users. It can be used as portfolio of designer or else... Any block with this type has special active areas with images which has hover effects. Functionality that will run after an user click on this elements you should implement by yourself.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit/ui-kit-projects/for-tpl/projects-2.html"

  4. LESS version:

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-projects/less/project-2.less';

  5. Component Modification:

    So about modifications which you can do.

    This block is very simple. You can replace all texts and images. For change backgrounds, please use the css. For example you can change background like this: 

    .project-2 {
                background-image: url("your image");
                }
