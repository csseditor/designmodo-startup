Block Footer #14

  1. Overview

    This block was developed as special footer block. All blocks that marked as footer should use once on one page.

  2. Installation

    If you haven't already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    To use this block, simply copy and paste the following rows into your HTML file: 

    include "../../../ui-kit/ui-kit-footer/for-tpl/footer-14.html"

  4. LESS version:

    Copy the same code as in css version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit/ui-kit-footer/less/footer-14.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    You can modify the elements in this block as you wish, but you should keep the same HTML structure to maintain its look and functionality.

    Like all the other blocks this block is based on a grid. It contains a logo, navigation links and social buttons. You need to add your own URLs to the social buttons.
