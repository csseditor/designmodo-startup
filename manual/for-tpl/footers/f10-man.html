Block Footer #10

  1. Overview

    This block was developed as a special footer block. All blocks that are marked as footer can only be used once on one page.

  2. Installation

    If you haven't already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    To use this block, simply copy and paste the following rows into your HTML file: 

    include "../../../ui-kit/ui-kit-footer/for-tpl/footer-10.html"

  4. LESS version:

    Copy the same code as in simple version and paste it in your '.less' file. Put the following line after that code: 

    @import '../../ui-kit/ui-kit-footer/less/footer-10.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    You can modify the elements in this block as you wish, but you should keep the same HTML structure to maintain its look and functionality.

    This block has three parts. The top part contains four columns with navigation links.

    The second part has a subscription form. It was developed as a form and you can send the data from it the server or do something else with it. In order to use the functionality of this form you need to add parameters to the form tag.

    The bottom part of this block contains a logo, addresses and a help section (you can replace it with anything you want).
