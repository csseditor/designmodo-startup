Block for creating footer. Codename “footer-11”.

  1. Overview

    This block was developed as special footer block. All blocks that marked as footer should use once on one page.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit-footer/for-tpl/footer-11.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-footer/less/footer-11.less';

    All versions:

    So about modifications which you can do.

    Any elements of this block you can modificate as you want, but you should save the html structure for saving design and functionality of this block.

    This block as any other based on grid. It contains logo, navigation and social buttons on top, slogon in the middle andh links in the bottom. For social buttons you should add your urls.
