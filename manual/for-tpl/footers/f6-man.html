Block Footer #6

  1. Overview

    This block was developed as a special footer block. All blocks that are marked as footer can only be used once on one page.

  2. Installation

    If you haven't already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    To use this block, simply copy and paste the following rows into your HTML file: 

    include "../../../ui-kit/ui-kit-footer/for-tpl/footer-6.html"

  4. LESS version:

    Copy the same code as in css version and paste it in your '.less' file. Put the following line after that code: 

    @import '../../ui-kit/ui-kit-footer/less/footer-6.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    You can modify the elements in this block as you wish, but you should keep the same HTML structure to maintain its look and functionality.

    This block has a logo, three columns with navigation links, a subscription form, and a container for social buttons <p class="social-btns">...</p>. You need to add your own URLs to the social buttons. The subscription form is just a text field and a button, so it doesn't have any functionality by default. You need to add your own JavaScript or wrap these elements with a real form tag with the parameters you need.
