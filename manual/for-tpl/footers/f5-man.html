Block Footer #5

  1. Overview

    This block was developed as special footer block. All blocks that marked as footer should use once on one page.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit/ui-kit-footer/for-tpl/footer-5.html"

  4. LESS version:

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit/ui-kit-footer/less/footer-5.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    Any elements of this block you can modificate as you want, but you should save the html structure for saving design and functionality of this block.

    All elements of this block have center alignment. This block has logo, slogan and Follow-button.
