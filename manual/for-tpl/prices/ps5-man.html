Block for creating prices. Codename “price-5”.

  1. Overview

    This type of blocks was created special for showing prices for your products to your users. All of these blocks based on grids and have design for comfort view of the price for your products.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit-price/for-tpl/price-5.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-price/less/price-5.less';

    All versions:

    So about modifications which you can do.

    This block has three columns with headers, description of the product and button that visible only in hover state of plan for choose it. Any content in these columns you can replace to another that you want, but you should save the html structure.

    Each description has button - the functionality of this button you should develop by youself. Any button has two states - normal and disabled. For make button disabled add class="disabled" to the button.
