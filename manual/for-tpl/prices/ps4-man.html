Block for creating prices. Codename “price-4”.

  1. Overview

    This type of blocks was created special for showing prices for your products to your users. All of these blocks based on grids and have design for comfort view of the price for your products.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit-price/for-tpl/price-4.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-price/less/price-4.less';

    All versions:

    So about modifications which you can do.

    This block has two parts. Left part has price-plan with header and description. All plan area is active button. The functionality of this button you should develop by youself. Content of the plan you can modificate as you want, but you should save the html structure.

    Right part - the description blocks. You can replace the content of each of this blocks, but you should save the html structure.
