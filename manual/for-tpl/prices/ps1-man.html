Block Price #1

  1. Overview

    This type of blocks was created special for showing prices for your products to your users. All of these blocks based on grids and have design for comfort view of the price for your products.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit/ui-kit-price/for-tpl/price-1.html"

  4. LESS version:

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit/ui-kit-price/less/price-1.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    This block has three columns with headers, description of the product and button for choose. Any of these column has special marker for informing user - it's the new product or not. Any content in these columns you can replace to another that you want, but you should save the html structure.

    If you want use the "NEW" marker functionality you should add <div class="ribbon">NEW</div> before closer tag of plan <div class="span4 plan">...(this)</div> . You can replace word "NEW" to another word.

    Each description has button - the functionality of this button you should develop by youself. Any button has two states - normal and disabled. For make button disabled add class="disabled" to the button.
