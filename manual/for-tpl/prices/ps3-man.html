Block Price #3

  1. Overview

    This type of blocks was created special for showing prices for your products to your users. All of these blocks based on grids and have design for comfort view of the price for your products.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit/ui-kit-price/for-tpl/price-3.html"

  4. LESS version:

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit/ui-kit-price/less/price-3.less';

  5. Component Modification:

    So about modifications which you can do.

    This block has three columns with headers, price block, description of the product and button for choose. Any content in these columns you can replace to another that you want, but you should save the html structure.

    Each description of plan has button - the functionality of this button you should develop by youself. Any button has two states - normal and disabled. For make button disabled add class="disabled" to the button.
