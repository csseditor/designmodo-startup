Block for creating header. Codename “header-7”.

  1. Overview

    This block was developed as special header block. All blocks that marked as header should use once on one page. But this block is very specific, you can use it as presentation. It doesn't has standart navigation bar.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit/ui-kit-header/for-tpl/header-7.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit/ui-kit-header/less/header-7.less';

    All versions:

    So about modifications which you can do.

    This block, as many others header blocks in our framework, has special place for background image. The place for background image - <div class=”background”>...</div> For add your image as background you should add CSS rulers for this element. And you can manage opacity of this element. For example as this: 

    .background {
    background-image:url(‘your image url’);
    opacity: opacity value;
}

    In ‘Less’ version for opacity property you can use cross browser function: 

    .opacity(value from 0 to 100);

    This header block is very specific and you can use it as presentation. So you can use it as wrapper for all your site content. It has Carousel. One slide - one page. You can change content for slides of this carousel. For this you should replace content inside <div class=”pt-page”> ... </div>

    This block has ‘.background’ tag for slides not for the main block because it has sense (you should have chance to change background for anyone of slides).

    But anyway you can use it as header... It means if you want place content after this block you can do it.
