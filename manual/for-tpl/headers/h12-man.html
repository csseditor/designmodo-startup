Block for creating header. Codename “header-12”.

  1. Overview

    This block was developed as special header block. All blocks that marked as header should use once on one page. This block has specific navigation bar that has only standart state, that you can see in documentation (at left side of your screen now).

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit-header/for-tpl/header-12.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-header/less/header-12.less';

    All versions:

    So about modifications which you can do.

    This block, as many others header blocks in our framework, has special place for background image. The place for background image - <div class=”background”>...</div> For add your image as background you should add CSS rulers for this element. And you can manage opacity of this element. For example as this: 

    .background {
    background-image:url(‘your image url’);
    opacity: opacity value;
}

    In ‘Less’ version for opacity property you can use cross browser function: 

    .opacity(value from 0 to 100);

    This block consist from two inner blocks. One of this it standart content block <div class="hero-unit">...</div>.

    Second block - <div class="aside">...</div> with promo and coast.
