Block Crew #4

  1. Overview

    This type of block was developed special for demonstrated crew of your company or project. Any block with this type has special active areas with images which has hover effects. Functionality that will run after an user click on this elements you should implement by yourself.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit/ui-kit-crew/for-tpl/crew-4.html"

  4. LESS version:

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit/ui-kit-crew/less/crew-4.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    This block has slogonon top of it and block with three columns that consits special blocks for showing members of your crew. You can modificate anyone of it, but you should remember that you can't reorganize the html structure if you want to leave functionality of this block as it exist now.

    For add images into project boxes you should insert your image between <div class="photo">...</div> .

    Each of these blocks has special information block <div class="info">...</div> you can place here information about the member.

    And yet anyone of it has social buttons inside the boxes that visible only in hover state, it situated between <div class="overlay">...</div> .
