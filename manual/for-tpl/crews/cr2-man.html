Block Crew #2

  1. Overview

    This block was specifically developed to show your company or project crew. All the crew blocks have special image areas with a hover effect. You need to implement the functionality behind the user click on these elements yourself.

  2. Installation

    If you haven't already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    To use this block, copy and paste the following line of code into your HTML file: 

    include "../../../ui-kit/ui-kit-crew/for-tpl/crew-2.html"

  4. LESS version:

    Copy the same code as in simple version and paste it in your '.less' file. Put the following line after that code: 

    @import '../../ui-kit/ui-kit-crew/less/crew-2.less';

    All versions:

    In this section you will see what modifications you can do to the block.

    This block has a slogan on the top and three columns which show the crew members in a special box. You can modify the info in any of these elements, but you should keep the same HTML structure to maintain its look and functionality.

    In order to add images into the boxes you need to insert your image between <div class="photo">...</div> .

    Each of these blocks has a special information block <div class="info">...</div>, where you can put information about the crew member. Each of these blocks also has social buttons on the bottom.
