Block Content #18

  1. Overview

    This block was developed as standard content block. Each block that marked as 'content' could be used multiple time in one page.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    
include "../../../ui-kit/ui-kit-content/for-tpl/content-18.html"

  4. LESS version:

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    
@import '../../ui-kit/ui-kit-content/less/content-18.less';

  5. Component Modification:

    So about modifications which you can do.

    This block has two rows. In first row - caruosel with center alignment an controls for which have absolute position.

    The second row has three columns with titles and description texts. You can modificate any element of this block, but you should save the html structure if you want to save view and functionality of it.

    Content of the carousel is situated between <div class="carousel-inner">...</div>. You can replace any of inner item and add new.
