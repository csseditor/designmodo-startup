It's standart content block. Codename “content-15”.

  1. Overview

    This block was developed as standard content block. Each block that marked as 'content' could be used multiple time in one page.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

    For using this block you should only paste next rows of code into your html file: 

    
include "../../../ui-kit-content/for-tpl/content-15.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    
@import '../../ui-kit-content/less/content-15.less';

    All versions:

    So about modifications which you can do.

    This block as any content block based on grid. It's simple html block. You can modificate any element of this block, but you should save the html structure if you want to save view and functionality of it.

    It contains the title with center alignment on top of it, text in two columns and in bottom row of this block situated image in left column and item with title and description in right column.
