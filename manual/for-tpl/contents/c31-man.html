Block Content #31

  1. Overview

    This block was developed as standard content block. Each block that marked as 'content' could be used multiple time in one page.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    
           include "../../../ui-kit/ui-kit-content/for-tpl/content-31.html"

  4. LESS version:

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    
           @import '../../ui-kit/ui-kit-content/less/content-31.less';

  5. Component Modification:

    So about modifications which you can do.

    This block as any content block based on grid. You can modificate any element of this block, but you should save the html structure if you want to save view and functionality of it.

    This block has special wrapper which use elements <div class="number">...</div> as animated list markers. It consists from another inner blocks. You can use anoter content blocks inside this block.

    This block has two elements - blocks which was developed special for it. First block has title and text on top of it and image in the bottom.

    The second block - caroucel. You can change content for slides of this carousel. For this you should replace content inside <div class=”pt-page”> ... </div>

    You can add the new blocks into this block, but you should use the same structure for any of iner blocks.

    Structure of inner block: 

    
            
    
                
    
                    
                    1
                
    
                
    
                    Your content...
