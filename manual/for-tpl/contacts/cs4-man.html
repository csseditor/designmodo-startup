Block Contacts #4

  1. Overview

    This block was developed to give the users the chance to contact you or your organization. All of the blocks of this type have links for your contacts in the form of the phone number or the email address.

  2. Installation

    If you haven't already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    To use this block, simply copy and paste the following rows into your HTML file: 

    include "../../../ui-kit/ui-kit-contacts/for-tpl/contacts-4.html"

  4. LESS version:

    Copy the same code as in simple version and paste it in your '.less' file. Put the following line after that code: 

    @import '../../ui-kit/ui-kit-contacts/less/contact-4.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    This block is very simple. It only has some social buttons, a title, some description text and a link for sending email. You can modify any of these element as you wish, but you should follow the same HTML structure to maintain the same look. You can change the background of this block through CSS.
