Block for creating contacts. Codename “contacts-4”.

  1. Overview

    This block was developed special for give to users chance to have contact with you or your organisation. All of this type of blocks have links for your contacts as phone, email and other.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit-contacts/for-tpl/contacts-4.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-contacts/less/contact-4.less';

    All versions:

    So about modifications which you can do.

    This block is very simple. It has only social buttons, title, description and link for sending email. All of these elements you can modificate as you want, but if you want thats it looks like now you should save the html structure. You can change the background of this block using css.
