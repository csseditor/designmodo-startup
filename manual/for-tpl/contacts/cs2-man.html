Block for creating contacts. Codename “contacts-2”.

  1. Overview

    This block was developed special for give to users chance to have contact with you or your organisation. All of this type of blocks have links for your contacts as phone, email and other.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit-contacts/for-tpl/contacts-2.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-contacts/less/contact-2.less';

    All versions:

    So about modifications which you can do.

    This block based on grid with left and right part. In left part - title or slogan and feedback form. The feedback form was developed as form which you can modificate if you need. And you should give some indificator for it, if you want use data from this form for send it to your server or do something else.

    In the right part of this block there are description - text that you can modificate as you want. This part has special links for phone and email.

    And yet it has special place for map <div class="map">...</div>. For using this functionality you should have map with your settings. This map you should add into this place as iframe... or make it somthig else, but you should know what you doing.
