Block for creating contacts. Codename “contacts-5”.

  1. Overview

    This block was developed special for give to users chance to have contact with you or your organisation. All of this type of blocks have links for your contacts as phone, email and other.

  2. Code

    Simple version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first

    For using this block you should only paste next rows of code into your html file: 

    include "../../../ui-kit-contacts/for-tpl/contacts-5.html"

    "LESS" version:

    if you not already done all instructions that you had to read on the home page of this documentation please make it first

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    @import '../../ui-kit-contacts/less/contact-5.less';

    All versions:

    So about modifications which you can do.

    This block was based on grid and it consist from two parts - top and bottom. The top part of this block has special place for map <div class="map">...</div>. For using this functionality you should have map with your settings. This map you should add into this place as iframe... or make it somthig else, but you should know what you doing.

    The second - bottom part consist from three columns with description. These columns have post adresses, and links to your FAQ section (you should have the page for FAQ) and link for sending email. You can modificate all content of these columns.
