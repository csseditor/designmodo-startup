Block Contacts #5

  1. Overview

    This block was developed to give the users the chance to contact you or your organization. All of the blocks of this type have links for your contacts in the form of the phone number or the email address.

  2. Installation

    If you haven't already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    To use this block, simply copy and paste the following rows into your HTML file: 

    include "../../../ui-kit/ui-kit-contacts/for-tpl/contacts-5.html"

  4. LESS version:

    Copy the same code as in css version and paste it in your '.less' file. Put the following line after that code: 

    @import '../../ui-kit/ui-kit-contacts/less/contact-5.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    This block is based on a grid and consists of two parts - top and bottom. There is a special area for a map in the top part of the block: <div class="map">...</div>. In order to use this functionality, you need to have a map in your settings. Simply add your map in this container as an iframe or use another technology if you know exactly what you are doing.

    The bottom part consist of three columns with descriptions. These columns have addresses and links to your FAQ section (you should have a page for FAQ). There is also a link for sending email. You can modify all of the content in these columns.
