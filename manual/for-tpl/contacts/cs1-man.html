Block Contacts #1

  1. Overview

    This block was developed special for give to users chance to have contact with you or your organisation. All of this type of blocks have links for your contacts as phone, email and other.

  2. Installation

    if you not already done all instructions that you had to read on the how to install page of this documentation please make it first

  3. CSS version:

    For using this block you should only paste next rows of code into your html file: 

    
include "../../../ui-kit-contacts/for-tpl/contacts-1.html"

  4. LESS version:

    you should paste same code (as in simple version) and paste next code into your '.less' file after that rows that you already had to paste. 

    
@import '../../ui-kit-contacts/less/contact-1.less';

  5. Component Modification:

    So about modifications which you can do.

    This block based on grid with left and right part. In left part - title or slogan and feedback form. The feedback form was developed as form which you can modificate if you need. And you should give some indificator for it, if you want use data from this form for send it to your server or do something else.

    In the right part of this block there are description - text that you can modificate as you want and an element <div class="phone">...</div> for main phone number with hours of work. It has special design. You can replace any information in this part. For it looks like now you should save html structure!
