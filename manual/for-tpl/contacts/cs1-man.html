Block Contacts #1

  1. Overview

    This block was developed to give the users the chance to contact you or your organization. All of the blocks of this type have links for your contacts in the form of the phone number or the email address.

  2. Installation

    If you haven't already done all the instructions from the how to install page of this documentation, please do that first.

  3. CSS version:

    To use this block, simply copy and paste the following rows into your HTML file: 

    include "../../../ui-kit/ui-kit-contacts/for-tpl/contacts-1.html"

  4. LESS version:

    Copy the same code as in css version and paste it in your '.less' file. Put the following line after that code: 

    @import '../../ui-kit/ui-kit-contacts/less/contact-1.less';

  5. Component Modification:

    In this section you will see what modifications you can do to the block.

    This block is based on a grid with left and right columns. The left column has a title or a slogan and a feedback form. The feedback form was developed as a form element which you can modify if you need to. You also need to have an indentificator for it, if you want to use the data from this form to send it to your server or do something else with it.

    The right column of the block has description text which you can modify as you wish, and an element <div class="phone">...</div> for the main phone number and availability hours. You can replace any of the information in this part, but you should follow the same HTML structure in order to maintain its look.
