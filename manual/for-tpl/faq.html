F.A.Q.

I'm pull the Startup framework from the repository and all of it components have a demaged view. What does it meen?

The Startup Framework based on the Flat-UI framework and it repository contains Flat-UI as project submodule. So this situation can appears if you pull the project without submodule. For fix it, you should execute thouse commands in your git console:

git submodule init

git submodule update

After this submodule folder 'flat-ui' mustn't be empty, and Startup components have to view right way.
