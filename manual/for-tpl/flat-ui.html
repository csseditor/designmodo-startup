Intergration with Flat UI

Startup framework based on Flat-UI version 2.0.0, so you can use all elements of this version inside it.

The same way you can buy the profesional version of the Flat-UI framework and use it instead of Flat-UI version 2.0.0.

For use Flat-UI Pro you should place it into flat-ui folder inside the project and remove from code all elements that marked as "Using only with Flat-UI (free)".
