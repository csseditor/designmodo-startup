How to install

Startup framework use HTML5 doctype! So your html file should start with next rows:

<!DOCTYPE html> <html lang="your site language"> ...

First you need have the package of Startup Framework or have access to it repository on the Github.

For use it the folder with Startup framework should be placed into root folder of your project (You can change it location, but you should be attentive with urls in this situation).

The Startup framework has two abstract versions which you can use. * Under the abstract versions we mean only logical parts. So in real we have one version.

The Simple version

If you have no experience developing with "LESS", then you can use this version.
For start developing with simple version you should know only html and css (and if you want do some more interesting effects - javascript).

The "LESS" version

If you want to use "LESS" it's your choise.
We made all to make usability of our framework simpler.

Code

  1. First step is equal to the simple version

  2. Second step. In your main '.less' file add next rows of code:

    @import '(way to the startup folder)/flat-ui/less/config.less';
@import '(way to the startup folder)/flat-ui/less/mixins.less';
/*only fo flat-ui free*/ @import '(way to the startup folder)/common-files/less/flat-ui-free2.0.0-fix.less'; /* end only for flat-ui free*/
@import '(way to the startup folder)/common-files/less/mixins.less';
@import '(way to the startup folder)/common-files/less/common.less';
@import '(way to the startup folder)/common-files/less/navbar.less';
@import '(way to the startup folder)/common-files/less/helper.less';

  3. The last step is equal to the simple version

Warning!

All components of Startup framework use responsive design. Some of it could have differences between browser that support JavaScript and browsers without JavaScript support. It's not critical difference. But if you will develop product for these browsers, please be attentive and don't forget check the result.
