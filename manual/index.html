Startup

We have a great surprise for Designmodo fans our first free HTML user interface kit.

It contains many basic and complex components which are great for designers to have at hand: buttons, inputs, button groups, selects, checkboxes and radio-buttons, tags.

How to use

Startup framework use HTML5 doctype! So your html file should start with next rows:

<!DOCTYPE html> <html lang="your site language"> ...

The Startup framework has two abstract versions which you can use. * Under the abstract versions we mean only logical parts. So in real we have one version.

The Simple version

If you have no experience developing with "LESS", then you can use this version.
For start developing with simple version you should know only html and css (and if you want do some more interesting effects - javascript).

The "LESS" version

If you want to use "LESS" it's your choise.
We made all to make usability of our framework simpler.

Code

  1. First step is equal to the simple version

  2. Second step. In your main '.less' file add next rows of code:

    @import '../../flat-ui/less/config.less';
@import '../../flat-ui/less/mixins.less';
/*only fo flat-ui free*/ @import '../../common-files/less/flat-ui-free2.0.0-fix.less'; /* end only for flat-ui free*/
@import '../../common-files/less/mixins.less';
@import '../../common-files/less/common.less';
@import '../../common-files/less/navbar.less';
@import '../../common-files/less/helper.less';

  3. The last step is equal to the simple version
